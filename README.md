# Flask Web Application

Учебное веб-приложение на Flask с авторизацией пользователей и управлением данными.

## Стек
- Python
- Flask
- SQLAlchemy
- SQLite
- Flask-SocketIO

## Функционал
- регистрация и авторизация пользователей
- сессии и личный кабинет
- CRUD операции для продуктов
- работа с базой данных (SQLAlchemy)
- JSON endpoint (/users)
- базовая работа с websocket
- шаблоны Jinja

## Запуск проекта

```bash
git clone https://github.com/KhayrullinTimur/flask-web-app.git
cd flask-web-app
pip install -r requirements.txt
python app.py
